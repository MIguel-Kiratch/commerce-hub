# Commerce Hub

Sistema de cadastro e gerenciamento de estabelecimentos desenvolvido em Python.

## Funcionalidades
- Cadastrar estabelecimento
- Listar estabelecimentos
- Alterar status (ativo/desativado)

## Tecnologias utilizadas
- Python
- Estruturas condicionais
- Listas e dicionários

## Como executar
1. Baixe o arquivo
2. Execute no terminal:
   python nome_do_arquivo.py
